# SOFTWARE REQUIREMENTS SPECIFICATION
# HOSPITAL MANAGEMENT SYSTEM
# Version 2.0

## 1. INTRODUCTION
### 1.1 Purpose
This document describes the requirements for the Hospital Management System.

## 2. FUNCTIONAL REQUIREMENTS

### 2.1 Patient Management Module
- Register new patients
- Update patient details
- View patient history

### 2.2 Doctor Management Module
- Register doctors
- Update doctor profiles
- View specializations

### 2.3 Appointment Management Module
- Book appointments
- Cancel appointments
- View doctor availability

### 2.4 Billing Module
- Generate bills
- Process payments
- Store receipts
