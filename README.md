# Hospital Management System (HMS)

Simple academic Hospital Management System for SCM practicals and project demo.

## Overview
Modules:
- Patient, Doctor, Appointment, Billing, Medical Records
- Staff, Inventory, Room Allocation, Emergency
- Reporting, Security & Backup

## Structure
- `SRS_Hospital_Management.md` — Software requirements (versions tracked)
- `docs/` — DFDs, diagrams, additional docs
- `diagrams/` — ER / UML / DFD images
- `sql/` — sample DB schema
- `sample_data/` — CSV sample data

## How to show this for the practical
1. Show commit history (Versions 1.0 → 5.0) from GitHub repo page.
2. Open `SRS_Hospital_Management.md` and point to the Version 5.0 section.
3. Show `diagrams/` and `docs/` files for supporting diagrams and DFDs.
